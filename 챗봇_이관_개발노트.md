## 1. 현재 챗봇의 핵심 아키텍처
현재 구현완료된 v2.0 챗봇은 일반적인 규칙(버튼형/키워드형) 챗봇이 아닌, **초거대 AI(Gemini)가 문맥을 이해하고 직접 텍스트를 판단하여 답변하는 "AI 두뇌 탑재 챗봇"** 입니다. 

1. **카카오톡 챗봇 플랫폼 (Kakao i Open Builder)**
   - 사용자의 카카오톡 텍스트 발화를 잡아내는 역할
   - **폴백 블록(순수 텍스트)**에 스킬을 적용하여 사용자의 모든 타이핑을 파이썬 서버로 포워딩.
2. **Cloudflare Tunnel (`cloudflared`)**
   - 로컬 PC(대장님 PC)의 `5055` 포트를 외부 카카오 서버도 접속할 수 있는 퍼블릭 주소(`https://---.trycloudflare.com`)로 뚫어주는 역할.
3. **Python FastAPI Backend Server (`kakao_bot_test.py`)**
   - 들어온 텍스트를 수신하고, Google Gemini API로 전송.
4. **Google Gemini API 엔진 (`gemini-2.5-flash`)**
   - 텍스트의 맥락을 분석하여 **단순 질문**인지, 혹은 제품에 문제가 생긴 **결함/파손/고장 상황**인지를 분류하여 그에 맞는 유연하고 능동적인(사진/영상 요구) 멘트를 자동 생성.

### STEP 3: 핵심 서버 코드 생성 (`kakao_bot_test.py`)
현재 최종 완성된 버전 2.0 코드를 그대로 복사해서 만듭니다.

```python
from fastapi import FastAPI, Request
import uvicorn
import os
from dotenv import load_dotenv

try:
    import google.generativeai as genai
except ImportError:
    genai = None

load_dotenv()

app = FastAPI(title="Kakao Chatbot Test v2.0 - AI Powered")

# Gemini API 모드 설정 (최신 gemini-2.5-flash 탑재)
GEMINI_API_KEY = os.getenv("GEMINI_API_KEY", "")
if GEMINI_API_KEY and genai:
    genai.configure(api_key=GEMINI_API_KEY)
    model = genai.GenerativeModel('gemini-2.5-flash')
else:
    model = None

@app.post("/api/kakao")
async def kakao_webhook(request: Request):
    try:
        body = await request.json()
        
        user_utterance = ""
        if "userRequest" in body and "utterance" in body["userRequest"]:
            user_utterance = body["userRequest"]["utterance"].strip()
            
        print(f"고객 발화 수신: '{user_utterance}'")
        
        reply_message = ""
        
        if model:
            # ----------------------------------------------------
            # [진행 완료] LLM 프롬프트 디자인 (의도 파악 및 분기)
            # ----------------------------------------------------
            prompt = f"""
            너는 쇼핑몰 또는 서비스 고객센터의 CS 응대 챗봇이야.
            너의 주된 목적: 고객의 말을 똑똑하게 이해하고, 
            기능 고장, 불량, 파손, 작동 이상 등 '문제가 있는 상황'이라고 판단되면 
            무조건 사진이나 영상을 첨부해달라고 정중하고 명확하게 우선적으로 요청해.
            만약 배송안내, 영업시간 같은 '단순 일반 질문'이면 사진을 묻지 말고, 일반적인 안내 멘트로만 대답해.
            친절하고 서비스 마인드가 돋보이는 1~2문단 길이의 존댓말로 대답해.
            
            고객의 말: "{user_utterance}"
            챗봇 답변:
            """
            
            try:
                response_ai = model.generate_content(prompt)
                reply_message = response_ai.text.strip()
                print("🧠 AI 응답 성공!")
            except Exception as e:
                print("Gemini API 호출 에러:", e)
                reply_message = "죄송합니다, AI 분석 중 문제가 발생했습니다. (혹시 제품에 문제가 있으시다면 사진/영상을 남겨주세요!)"
                
        else:
            # 환경변수 세팅 실패 시 대비용(Fallback) 로직
            reply_message = "현재 AI 두뇌 키(API)가 인식되지 않아 답변이 불가능합니다. 관리자에게 문의하세요!"

    except Exception as e:
        print("데이터 파싱 에러:", e)
        reply_message = "알 수 없는 시스템 오류가 발생했습니다."
    
    # 카카오톡 스킬 응답(JSON) 규격 포맷팅
    response = {
        "version": "2.0",
        "template": {
            "outputs": [
                {
                    "simpleText": {
                        "text": reply_message
                    }
                }
            ]
        }
    }
    return response

if __name__ == "__main__":
    print("==================================================")
    print("🤖 카카오톡 AI 챗봇 서버가 구동 중입니다. (v2.0)")
    print("포트: 5055 | 웹훅 경로: /api/kakao")
    if model:
        print("✅ 구글 Gemini AI 엔진 시스템 On")
    else:
        print("⚠️ Gemini API 키가 누락되었습니다.")
    print("==================================================")
    
    uvicorn.run(app, host="0.0.0.0", port=5055)
```
